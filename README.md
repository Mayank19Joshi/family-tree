# Joshi Family Tree

This is a static website. You can open `index.html` locally or publish it on any static host.

## Best permanent hosting options

### GitHub Pages
1. Create a GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and any future assets.
3. In the repository settings, open `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/root`.
6. Your site will be published on a stable GitHub Pages URL.

### Netlify
1. Create a Netlify account.
2. Create a new site from drag-and-drop or from a Git repository.
3. Upload this folder or connect the repo.
4. Because this is a static site, no build command is required.
5. Netlify will give you a stable URL and lets you attach a custom domain.

## Files to publish

- `index.html`
- `styles.css`
- `script.js`
- `family tree.txt` is optional source data and does not need to be public
